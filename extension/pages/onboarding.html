Welcome to ProtoConsent

ProtoConsent puts you in control of how websites use your data. Choose a default privacy profile below. You can always change it or customize individual sites from the toolbar icon.

What ProtoConsent does for you

Consent Control
Your profile blocks tracking, ads, and more for every site you visit.
Enhanced Protection
Optional third-party blocklists add up to 1.8M extra tracking rules.
GPC Signal
A Global Privacy Control signal is sent to websites automatically when tracking is blocked.
Log & Whitelist
Monitor blocked requests and allow specific domains if needed.
Per-site Control
Override your default profile for specific websites.
Fingerprint Protection
Strips high-entropy Client Hints headers that websites use to identify your device.
Website Transparency
Checks .well-known/protoconsent.json declarations to verify a site's stated data practices.

Enhanced list updates

ProtoConsent can check for enhanced list updates from an open-source repository (ProtoConsent/data on GitHub). This happens over the network. No data is sent.

What it does
Fetches the latest blocklist metadata from a public GitHub repository. New tracker domains are added regularly by the community.
Your control
You can change this at any time in Settings. When disabled, only the lists bundled with the extension are used.
