Detect which consent platforms are present on each page and observe what their cookies say. Results appear in the Log tab. Passive - does not modify anything.

Advanced
Consent cookies include an identifier. Leave empty (recommended): a unique random ID is generated for each page visit, so sites cannot link your visits. Set a fixed UUID only if a specific site requires it - the same ID will be reused across all visits, allowing sites to recognize you.
